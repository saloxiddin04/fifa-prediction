# ⚽ FIFA Position Prediction

AI-powered football player position prediction system using Machine Learning.

## 🚀 Quick Start

### Backend
\`\`\`bash
cd backend
python -m venv venv
venv\Scripts\activate  # Windows
pip install -r requirements.txt
python run.py
\`\`\`

### Frontend
\`\`\`bash
cd frontend
npm install
npm run dev
\`\`\`

## 📊 Model Performance

- **Field Player Model**: 76.53% accuracy
- **Goalkeeper Model**: 96.30% accuracy

## 🛠️ Tech Stack

- **Backend**: FastAPI, Scikit-learn, Pandas
- **Frontend**: React, Vite, TailwindCSS
- **ML**: Random Forest Classifier
